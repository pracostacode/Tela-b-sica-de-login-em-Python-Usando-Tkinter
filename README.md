# User Registration System

## Project Overview
This project is a user registration system built using Python and Tkinter, a popular GUI toolkit. It allows users to register their details and securely store them.

## Features
- User-friendly interface
- Input validation
- Secure storage
- Data retrieval

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/pracostacode/Tela-b-sica-de-login-em-Python-Usando-Tkinter.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Tela-b-sica-de-login-em-Python-Usando-Tkinter
   ```
3. Run the application:
   ```bash
   python main.py
   ```

## Dependencies
- Python 3.x
- Tkinter
